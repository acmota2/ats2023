## Geração de inputs

Intruções:

* abrir o ficheiro [InputGenerator.hs](./InputGenerator.hs) no interpretador de Haskell de uma das seguintes formas, consoante a instalação foi feita em Stack ou Cabal:

    * Stack:
        ```
        stack ghci InputGenerator.hs
        ```
    * Cabal:
        ```
        ghci InputGenerator.hs
        ```

* uma vez no interpretador, executar a função `main`:

    ```
    ghci > main
    ```

* um ficheiro chamado `test.txt` será gerado nesta pasta
